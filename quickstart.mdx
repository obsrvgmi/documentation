## Market Intelligence
Tools to analyze price action and track specific assets before you trade.

| Command | Description | Example |
| :--- | :--- | :--- |
| **/markets** | Displays a dashboard of all listed trading pairs on Decibel (e.g., APT-USD, BTC-USD). Shows live prices, 24h volume, and daily percentage changes. | `/markets` |
| **/chart** | Generates a real-time candlestick chart for the requested asset inside Telegram. Toggle timeframes (15m, 1H, 4H) to analyze trends. | `/chart APT` |
| **/watchlist** | Displays your saved list of favorite **Token Pairs**. Use this to filter out noise and only view prices for the assets you actively trade. | `/watchlist` |
| **/alert** | Sets a custom price alert. The bot will DM you immediately when the token hits your target price. | `/alert ETH 2800` |

## Trading Execution
Core commands for executing, managing, and closing your positions.

| Command | Description | Example |
| :--- | :--- | :--- |
| **/limit** | Opens the Limit Order interface. Allows you to set precise entry prices to buy (Long) or sell (Short) at a specific level. | `/limit BTC 65000` |
| **/positions** | Fetches a summary of all your currently active positions. View your entry price, liquidation price, and real-time PnL (ROI%). | `/positions` |
| **/close** | The quick-exit tool. Brings up a menu to close specific positions, or options to close all at market price. | `/close SOL` |
| **/orders** | Manages your "Open Book." View all pending limit orders or triggered stop orders. Cancel specific orders or wipe the book. | `/orders` |

## Risk Management
Protect your capital and automate your exits.

| Command | Description | Example |
| :--- | :--- | :--- |
| **/tp** | Adds or edits a **Take Profit** trigger on an open position. You can set this by price target or by percentage gain. | `/tp ETH 3500` |
| **/sl** | Adds or edits a **Stop Loss** trigger. Automatically closes your trade if the price moves against you to prevent liquidation. | `/sl APT 8.50` |

## Social & PVP
Features for tracking other traders, joining teams, and competitive interaction.

| Command | Description | Example |
| :--- | :--- | :--- |
| **/track** | Adds a specific trader's wallet address to your personal tracking list. Receive notifications on their activity. | `/track 0x123...abc` |
| **/untrack** | Removes a wallet from your tracking list. Stops the bot from sending notifications for that user. | `/untrack 0x123...abc` |
| **/guild** | Opens the Guild interface. Create your own trading team, join an existing guild, or view the leaderboard. | `/guild` |
| **/taunt** | Sends a playful or competitive message to another user. Best used after you win a trade against them. | `/taunt @username` |

## System & Settings

| Command | Description | Example |
| :--- | :--- | :--- |
| **/settings** | Access the global bot configuration. Adjust your default leverage, gas fee preset, and "One-Click Trading" mode. | `/settings` |
| **/wallet** | Checks your Aptos wallet balance and connects/disconnects your address. Shows available collateral. | `/wallet` |

## Automated Strategy Commands
*These commands enable advanced automated trading strategies. Please refer to their specific documentation pages for detailed setup instructions.*

| Command | Description | Example |
| :--- | :--- | :--- |
| **/copytrade** | **Follow the Winners.** Automatically mirrors the trades of a specific wallet using fixed or proportional sizing. | `/copytrade 0xABC...123` |
| **/countertrade** | **Fade the Losers.** The inverse of copy trading. Automatically opens the *exact opposite* position of the target wallet. | `/countertrade 0xXYZ...789` |
