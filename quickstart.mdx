## Market Data & Charts
Analyze the market before taking a position.

| Command | Description |
| :--- | :--- |
| **/markets** | Displays a snapshot of all available trading pairs on Decibel, including current prices, 24h volume, and daily percentage changes. |
| **/chart** `[token]` | Generates a real-time candlestick chart for the specified token. You can toggle between timeframes (15m, 1h, 4h, 1d) directly within the chat. |
| **/alert** `[token]` | Sets a price alert. The bot will DM you immediately when the specified token hits your target price, so you never miss a move. |

## Trading Execution
Commands for executing market, limit, and unique PVP strategy orders.

| Command | Description |
| :--- | :--- |
| **/long** / **/short** | Opens a market order. If no token is specified, it opens the "Quick Trade" menu. Reply to any token ticker with these commands for instant execution. |
| **/limit** `[token]` | Opens the Limit Order interface. Set specific entry prices to buy or sell only when the market hits your desired level. |
| **/copytrade** `[wallet]` | Auto-copy a target wallet. You can set a fixed amount per trade or mirror their leverage. When they win, you win. |
| **/countertrade** `[wallet]`| **(PVP Exclusive)** The inverse of copy trading. Automatically opens the *opposite* position of the target wallet. Use this if you think a specific trader is always wrong. |
| **/positions** | Views all active open positions. Includes live PnL, liquidation prices, and "Close Position" buttons. |
| **/close** | Closes positions. Use `/close all` to exit everything immediately, or `/close [token]` for specific assets. |
| **/orders** | Manages pending limit or trigger orders. View your open book and cancel orders individually or in bulk. |

## Risk Management
Protect your capital and automate exits.

| Command | Description |
| :--- | :--- |
| **/tp** `[token]` | Sets **Take Profit** triggers. Configure partial take profits (e.g., "Sell 50% of position at $2000") to lock in gains while letting the rest ride. |
| **/sl** `[token]` | Sets **Stop Loss** triggers. Protects against liquidation by automatically closing your position if the price moves against you by a set percentage. |

## System & Settings

| Command | Description |
| :--- | :--- |
| **/wallet** | Checks your Aptos wallet balance and connects/disconnects your address. Shows available collateral for trading. |
| **/settings** | Configures global bot preferences: Default leverage, slippage tolerance, notification settings, and "One-Click Trade" mode. |
